### GIT

`git init path` - create repository

`git add path` - add file to staging area (index)

`git commit -m "message"` - commit changes from index

`git restore path` - removes all changes that are not in index

`git reset --hard commit_id` - returns to that commit_id (removes new changes)

`git remote add remote_name url` - add new remote repo

`git push remote branch_name` - push commited changes

`git pull remote branch_name` - pull remote repo

`git branch branch_name` - create new branch

`git branch -d branch_name` - delete branch

`git checkout branch_name` - switch branch

`git merge branch_name` - merge branch into current branch
