### MS-SQL

create database

``` tsql
CREATE DATABASE [database];
```

delete table

``` tsql
DROP TABLE [table];
```

insert row

``` tsql
INSERT INTO [table] (column, ...)
VALUES (value, ...);
```

edit row

``` tsql
UPDATE [table] SET [column] = [value];
```

delete rows

``` tsql
DELETE FROM [table];
```
